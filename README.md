# Basic Text Analyzer - Initiation Fact-Checking

Contexte :Ce mini-projet a été réalisé dans le cadre de ma montée en compétences en programmation Python et en Intelligence Artificielle.

Objectif :
Développer un script basique d'analyse de texte (Natural Language Processing simplifié) capable de repérer un vocabulaire sensationnaliste ou clickbait, souvent associé à la propagation de fausses informations (désinformation).

Fonctionnalités :
- Nettoyage basique des données (passage en minuscules).
- Comparaison d'un narratif avec un lexique de mots-clés suspects prédéfinis.
- Génération d'une alerte et d'une recommandation pour l'analyste si le texte nécessite un fact-checking approfondi.

Technologies :Python
